visibles.
Realizar un pull (desde git local => abrir la consola [desde donde esta el repositorio local] => git pull [trae los cambios] )
7. Vuelva a modificar el archivo desde la interfaz web de Github, y antes de hacer el pull,
modifique algo DISTINTO en su copia local y haga un commit de dichos cambios.

8. Ahora pruebe hacer un push de sus cambios. ¿Qué sucedió? ¿ cómo puede proseguir?
Tenemos que resolver los problemas de los cambios. Elegir el cambio: 
Aceptar cambio actual | Aceptar combio entrante | Aceptar ambos cambios | Comparar cambios.
